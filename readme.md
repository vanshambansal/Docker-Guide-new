## 🚀 Run Locally

### Prerequisites
- Python 3.8+
- Git

### Quick Start

```bash
# Clone repository
git clone https://github.com/vanshambansal/Docker-Guide-new
cd Docker-Guide-new

# Install dependencies
pip install -r requirements.txt

# Start development server
mkdocs serve

# Open http://localhost:8000
```

### Build Static Site

```bash
# Build for production
mkdocs build

# Output in site/ folder
```
